TModuleType = (...);

Application module type: game module, gui app, no gui app or console app For backward compatibility mtGDX=-1 and others order by id.

Values
  • mtGDX = -1: Not implement yet
  • mtGUI = 0: Basic android application with visual activity(ies)
  • mtNOGUI = 1: May be daemons? Not implement yet.
  • mtCONSOLE_EXEC_APP = 2: Native console app. Write on FPC, for execute need 'Termux' and others.
  • mtGENERIC_LIB = 3: Native lib, may be used in another application types as 'so'
